# 组件列表

包含常用组件、gbeata 功能形组件、2D/3D 组件、数据可视化组件等

目前提供的组件如下：
## 功能组件

- [Loading 加载组件](/components/loading) - 优雅的加载动画组件
- [FlowEditor 流程编辑器](/components/floweditor) - 基于 react-flow 的可视化流程编辑
- [2D 画板](/components/2d) - 2D 绘图组件
- [Animate 动画](/components/animate) - 动画效果组件
- [DrawBoard 绘图板](/components/drawboard) - 交互式绘图板

## 3D 可视化组件

- [点云渲染](/components/point-cloud) - 海量点云分块渲染，支持 5000 万点
- [海量点云 LOD](/components/point-cloud-lod) - 八叉树 LOD 技术，支持 2 亿级点云
- [点云网格化](/components/point-cloud-mesh) - Marching Cubes 网格重建
- [机器人编辑器](/components/robot-editor) - 3D 模型交互编辑器
